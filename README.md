Personal AI Assistant

Assistente de IA desenvolvido com Python, Streamlit e integração com API Grok.

Funcionalidades

* Interface interativa via navegador
* Comunicação com modelo de IA
* Respostas em tempo real

Tecnologias utilizadas

* Python
* Streamlit
* Requests
* API Grok (xAI)

Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
```
Configuração

Crie um arquivo `.env` na raiz do projeto:

```
GROQ_API_KEY=sua_chave_aqui
```

Objetivo

Este projeto foi desenvolvido com foco em aprendizado prático, integração com APIs de IA e construção de portfólio.
